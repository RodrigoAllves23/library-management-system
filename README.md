# 📚 Library Management System

Sistema de gerenciamento de biblioteca escolar desenvolvido com:

- Java
- Spring Boot
- PostgreSQL
- Docker
- Swagger
- Spring Security

## Funcionalidades

✔ Cadastro de alunos  
✔ Cadastro de livros  
✔ Controle de empréstimos  
✔ Controle de devoluções  
✔ Sistema de reservas  
✔ Multas por atraso  

## Arquitetura

Arquitetura em camadas:

Controller → Service → Repository → Database
